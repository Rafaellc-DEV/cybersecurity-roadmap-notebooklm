# 🛡️ Cybersecurity & Pentesting Roadmap



Este repositório é um guia prático para quem deseja **trilhar carreira em Cibersegurança**, equilibrando:

- **Red Team (Ofensiva/Pentesting)**  

- **Blue Team (Defensiva/SOC/IR)**  

- **GRC (Governança, Risco e Compliance)**  



## 🎯 Objetivos

- Estruturar trilhas de aprendizado em diferentes áreas da segurança.  

- Detalhar o ciclo de vida de um **Pentest**.  

- Consolidar frameworks globais como **MITRE ATT&CK**, **OWASP Top 10** e **NIST CSF**.  



## 📚 Fontes

- NIST Cybersecurity Framework 2.0  

- MITRE ATT&CK Matrix  

- OWASP Top 10 (2024)  

- PTES (Penetration Testing Execution Standard)  

- Diretrizes do BACEN (Resolução 4.893)  



## 🚀 Roadmap

1. **Red Team:** Reconhecimento → Exploração → Pós-exploração → Relatório.  

2. **Blue Team:** Monitoramento → Detecção → Resposta a Incidentes.  

3. **Purple Team:** Integração contínua entre ofensiva e defensiva.
